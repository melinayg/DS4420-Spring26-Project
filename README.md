# Predicting and Understanding Coral Reef Bleaching Using Environmental Drivers

By Melina Yang and Arshia Mathur



## Overview

This project investigates coral reef bleaching prediction using two machine learning approaches. First, a Multilayer Perceptron (MLP) will be applied to classify whether a reef experiences bleaching based on environmental conditions at the time of survey, including sea surface temperature anomalies, turbidity, cyclone frequency, and degree heating weeks. Then, a Bayesian Logistic Regression model will be implemented to quantify which environmental stressors drive bleaching outcomes, with credible intervals on each predictor.
